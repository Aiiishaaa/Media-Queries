Modifier le fichier style.css de sorte que:
- au dessus de 480px de largeur d'écran:
	* on affiche les 3 colonnes sur la même ligne
	* le menu est affiché en ligne.
	
- au dessous de 480px de largeur d'écran
	* on n'affiche qu'une seule colonne par ligne, avec les 3 lignes qui se suivent
	* on affiche les 4 items du menu les uns en dessous des autres (un seul pr ligne)
	

Votre code DOIT être mobile-first.
Par défaut, le code fourni vous montre l'afichage final pour une haute résolution (au dessus de 480px), sans utilisation de media-queries.
Vous devez donc utiliser obligatoirement les media-queries pour réaliser cet exercice et rendre votre page mobile-first.
